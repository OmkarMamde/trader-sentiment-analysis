# Trader Performance vs Market Sentiment

## Objective
Analyze how market sentiment (Fear vs Greed) affects trader behavior and performance.

## Steps Performed
- Data cleaning and preprocessing
- Dataset merging based on date
- Feature engineering
- Exploratory data analysis
- Trader segmentation

## Key Insights
1. Trader profitability is highest during Extreme Greed periods.
2. Trade sizes tend to increase during Fear periods.
3. Market sentiment influences trading behavior and risk-taking.

## Strategy Recommendations
- Reduce leverage during Fear periods due to high volatility.
- Increase trading activity cautiously during Extreme Greed periods.

## Tools Used
Python, Pandas, Seaborn, Matplotlib
